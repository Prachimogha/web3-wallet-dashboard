# Web3 Wallet Dashboard

A Web3 dashboard that connects to MetaMask and displays wallet information.

## Features
- Connect MetaMask Wallet
- Show Wallet Address
- Display ETH Balance
- Show Network Information
- Disconnect Wallet

## Tech Stack
- React.js
- Ethers.js
- Tailwind CSS
- Web3

## Installation

1. Clone the repository
git clone https://github.com/Prachimogha/web3-wallet-dashboard.git

2. Install dependencies
npm install

3. Run project
npm start
